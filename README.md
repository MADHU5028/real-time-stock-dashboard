## Description
A real-time stock market dashboard built using Python and Streamlit.  
The application fetches live stock market data and visualizes it using interactive charts and technical indicators.

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- yFinance

## Features
- Select stock symbol
- Live stock price visualization
- Moving Average indicator
- RSI (Relative Strength Index)
- Interactive charts
- Manual data refresh

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
